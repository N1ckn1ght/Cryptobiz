# Cryptobiz

Приложение, отслеживающее криптовалюту Etherium.

Сохраняет котировки автоматически раз в 10 минут с помощью Foreground Service.

Можно смотреть курс в лайв-режиме, обновлять по кнопке Refresh и сохранять по кнопке SAVE.

Кнопка становится неактивной, если данные не обновлены через REFRESH и/или уже сохранены.

База данных имплементирована через Room.

<img src="https://i.imgur.com/6KOMuxj.jpg">

На главном экране расположены котировки в разных валютах, в RecyclerView внизу только в рублях из записей базы данных.

Слева от котировки дата и время, справа - изменение относительно предыдущей записи.

Пустое поле означает слишком незначительное изменение (~0.0000%).
