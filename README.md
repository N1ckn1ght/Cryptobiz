# Cryptobiz

Приложение, отслеживающее криптовалюту Etherium.

Сохраняет котировки автоматически раз в час с помощью Worker.

Можно смотреть курс в лайв-режиме и сохранять по кнопке SAVE.

Кнопка становится неактивной, если данные не обновлены через REFRESH и/или уже сохранены.

База данных имплементирована через Room.

<img src="https://i.imgur.com/AyxLtkH.jpg">

На главном экране расположены котировки в разных валютах, в RecyclerView только в рублях.

Слева от котировки дата и время, справа - изменение относительно предыдущей записи.

Пустое поле означает слишком незначительное изменение (~0.00%).
